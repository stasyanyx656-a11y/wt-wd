# Лабораторна робота № 14 - Розгортання CMS WordPress

Ця папка містить конфігурацію для автоматичного розгортання повноцінного середовища WordPress за допомогою **Docker Compose**. Це професійний підхід до роботи з CMS, який не вимагає ручного встановлення XAMPP/OpenServer.

## Як запустити сайт "AeroTech Hub"

1. **Вимоги**: У вас має бути встановлений [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Відкрийте термінал у цій папці (`lab14`).
3. Виконайте команду:
   ```bash
   docker-compose up -d
   ```
4. Зачекайте 1-2 хвилини, поки завантажаться образи бази даних MySQL та WordPress.
5. Відкрийте браузер і перейдіть за адресою: 
   **[http://localhost:8000](http://localhost:8000)**

## Налаштування при першому запуску

При першому відкритті сайту WordPress запропонує вам обрати мову (Українська) та створити обліковий запис адміністратора. 
Після цього ви зможете увійти в адмін-панель за адресою:
**[http://localhost:8000/wp-admin](http://localhost:8000/wp-admin)**

> **Підказка:** Щоб зупинити роботу сервера, виконайте в терміналі команду `docker-compose down`.
